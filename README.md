# Fundon Backend

Server for Fundon Application made using MongoDB , Node Js , Express in MVC ( Model View Controller ) Architecture .

## Server

[Server Repo](https://github.com/AnshTandon0/CodeQBackend)

### NodeJs Server dependencies used :

* Nodemon
* Mongoose
* Express
* jsonwebtoken (jwt)
* Nodemailer
